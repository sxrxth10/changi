BASE URL http://65.0.97.155:8000

Post/chat

📄 Description:
Ask a question related to Changi Airport.
This endpoint uses a Retrieval-Augmented Generation (RAG) pipeline backed by Pinecone and OpenAI.

Request Body (JSON)
{
  "query": "What are the shops available at Terminal 3?"
}

Sample Request
response = requests.post(
            "http://65.0.97.155:8000/chat",  
            json={"query": user_input}
        )

✅ Success Response:
Code: 200 OK
Content:
{
  "response": "You can find a variety of shops including luxury brands, electronics, and souvenirs in Terminal 3.",
  "sources": [
    "https://www.changiairport.com/en/maps.html",
    "https://www.jewelchangiairport.com/en/dine.html"
  ]
}

🛠️ Example curl Request:
curl -X POST http://<YOUR-EC2-IP>:8000/chat \
  -H "Content-Type: application/json" \
  -d '{"query": "Where is Terminal 1?"}'
